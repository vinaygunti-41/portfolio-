Personal Portfolio Website | HTML, CSS & JavaScript

📌 Project Overview

This Personal Portfolio Website is a responsive and interactive web application developed using HTML, CSS, and JavaScript. It serves as a digital portfolio to showcase my skills, projects, certifications, education, and professional achievements in Data Analytics and Web Development.

The portfolio is designed with a modern user interface, smooth animations, and responsive layouts to provide an engaging user experience across desktop, tablet, and mobile devices.

🎯 Objectives

Showcase professional profile and technical skills.
Present projects and certifications in an organized format.
Provide recruiters and employers with easy access to portfolio details.
Create a personal brand through a visually appealing web presence.
🛠️ Technologies Used
HTML5 – Structure and content organization.
CSS3 – Styling, layouts, and responsive design.
JavaScript – Interactive elements and animations.
Font Awesome – Icons and visual enhancements.
GitHub Pages – Website deployment and hosting.
✨ Key Features
Responsive design for all screen sizes.
Animated hero section.
Interactive navigation menu.
About Me section.
Skills showcase with icons.
Projects gallery with live links.
Certifications section.
Contact information and social media links.
Smooth scrolling and transition effects.
Modern UI/UX design.
